# 🎨 Portfolio Project


UI/UX 디자이너 **최지원**의 개인 포트폴리오 사이트입니다. 사이트 리뉴얼 프로젝트를 한눈에 볼 수 있도록 원페이지 구조로 제작하였으며, 각 프로젝트의 기획서·GitHub·최종 결과물로 바로 이동할 수 있습니다.

🔗 배포 사이트: https://jw123123.github.io/portfolio/

---

## 📖 프로젝트 개요

### 프로젝트 목적

여러 프로젝트의 결과물이 흩어져 있어 작업의 흐름과 역량을 한 번에 전달하기 어려웠습니다.

이에 프로젝트 소개부터 상세 결과물, 기술 스택, 자기소개까지 하나의 흐름으로 경험할 수 있는 원페이지 포트폴리오를 HTML/CSS/JavaScript 기반으로 구현하였습니다.

### 프로젝트 목표

- 프로젝트 중심의 직관적인 탐색 구조 설계
- 스크롤 인터랙션 기반 몰입형 UI 적용
- 영상 미리보기로 각 프로젝트의 결과물을 빠르게 전달
- 디자이너로서의 브랜드 아이덴티티 표현

### 프로젝트 방식

- 개인 프로젝트 (100%)
- Figma를 활용한 UI 설계
- HTML/CSS/JavaScript 기반 퍼블리싱
- GSAP(ScrollTrigger)을 활용한 스크롤 인터랙션
- Git을 통한 버전 관리

## 🌱 Learn & Future

### 프로젝트를 통해 배운 점

GSAP ScrollTrigger를 활용해 스크롤 기반 인터랙션을 직접 설계하며, 시각적 연출과 사용성 사이의 균형을 고민하는 방법을 익혔습니다. 또한 여러 프로젝트를 하나의 흐름으로 정리하면서 정보 구조 설계의 중요성을 배울 수 있었습니다.
